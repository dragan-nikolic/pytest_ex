## Tips & Tricks

### How to ativate venv on different platforms

* Linux/Mac
  * `$ source venv/bin/activate`
* Windows
  * Command: `> venv/Scripts/activate.bat`
  * PowerShell: `> venv/Scripts/Activate.ps1`
